## §Z — North Star: AI Visuals for Musicians

### 한 줄 positioning

> **YAGI = AI Visuals for Musicians**
>
> Musicians (가수 / K-pop label / agency / indie) 가 본인 음악 IP 의 visual
> output (뮤직비디오 / 콘셉트 영상 / 컴백 콘텐츠 / fan engagement 영상) 을
> AI 도구 + curated creator network 로 *제작 + 유포* 할 수 있는 platform.

### 시장 framing

기존 K-pop 뮤비 시장:
- 단일 official 뮤비 = 5천만원~1억원 production budget
- 3-6개월 production cycle (콘셉트 → 촬영 → 후반 → publish)
- 1 채널 (가수 official YouTube) deploy
- ROI metric = 단일 뮤비 view 수

야기 모델:
- Marketing budget 단가 (기존 시장 대비 낮거나 비슷)
- 1-2주 campaign cycle (brief → 다수 creator 응모 → 검수 → 유포)
- N AI 뮤비 + multi-channel (TikTok / IG / YouTube Shorts / X) 유포
- ROI metric = aggregate view + algorithm 보상 + creator network cross-pollination
- Repeat purchase = 곡 release / 컴백 / 콘셉트 별 반복 = **B2B SaaS revenue**

→ 야기 unique value = K-pop 시장 의 visual output 의 *양 + 채널 + 속도*
변화. "Costco for K-pop visuals" 또는 "Distributed video studio for music IP".

### Primary / Secondary / Tertiary customers

| Tier | Customer | 사용 product |
|---|---|---|
| **Primary** | Musicians (가수 본인 / K-pop label / agency / indie 음악인) | Phase 6 [새 프로젝트 시작] (1:1 commission 뮤비) + Phase 7 [+ 캠페인 요청] (1:N distributed campaign) |
| **Secondary** | Brand clients (음악 사용 광고, 앨범 컬래버) | Phase 10 [브랜드 협업 제안] (Inbound Track, deferred) — 광고 기반 |
| **Tertiary** | Self-Sponsored Artist Ad (Type 2) / External Brand Boost (Type 3) | Phase 6 Briefing Canvas Step 3 의 외부 광고주 toggle |

Primary 80% / Secondary 15% / Tertiary 5% 가 야기 사업 비중 design 의도.
